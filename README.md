libgnomekbd
==========

libgnomekbd is a library to view and test keyboard layouts.

libgnomekbd requires:

   * glib >= 2.44
   * gio >= 2.44
   * gobject >= 2.44
   * gobject-introspection
   * gtk+-3.0 >= 2.91.7
   * libxklavier >= 5.2
   * x11

Building is done with [Meson](https://mesonbuild.com).

Bugtracker:
   https://gitlab.gnome.org/GNOME/libgnomekbd/-/issues
